# vuejs-cli-playground

Playground baru, sekarang lewat cli/node. Karena playground sebelumnya terasa cumbersome.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
